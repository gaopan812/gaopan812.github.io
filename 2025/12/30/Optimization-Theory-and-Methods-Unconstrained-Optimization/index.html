最优化理论与方法-无约束优化

发表于 2025-12-30 更新于 2026-01-04 分类于 最优化

无约束优化

  • 无约束优化问题

    • 无约束优化问题是指在没有任何限制条件下，最小化一个依赖于实变量的目标函数：

      min⁡xf(x)\min_{x} f(x) xmin​f(x)

      其中 x∈Rnx \in \mathbb{R}^nx∈Rn（n≥1n \geq 1n≥1）是一个实向量，函数 f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R 通常是光滑的（例如连续可微）。

  • 什么是一个解？

    • 全局最小值点：
      点 x∗x^*x∗ 称为函数 fff 的全局最小值点，如果

      f(x∗)≤f(x),∀ x∈Rn.f(x^*) \leq f(x), \quad \forall\, x \in \mathbb{R}^n. f(x∗)≤f(x),∀x∈Rn.

    • 局部最小值点：
      大多数优化算法只能找到局部最小值点，即在某个邻域内使目标函数值最小的点。
      点 x∗x^*x∗ 称为局部最小值点（或弱局部最小值点），如果存在 x∗x^*x∗ 的一个邻域 N\mathcal{N}N，使得

      f(x∗)≤f(x),∀ x∈N.f(x^*) \leq f(x), \quad \forall\, x \in \mathcal{N}. f(x∗)≤f(x),∀x∈N.

    • 严格局部最小值点：
      若存在邻域 N\mathcal{N}N，使得

      f(x∗)<f(x),∀ x∈N,  x≠x∗,f(x^*) < f(x), \quad \forall\, x \in \mathcal{N},\; x \neq x^*, f(x∗)<f(x),∀x∈N,x=x∗,

      则称 x∗x^*x∗ 为严格局部最小值点。

  • 识别一个局部最小

    • 当目标函数 fff 光滑时，有多种高效且实用的方法可用于识别局部极小值点。

    • 特别地，若 fff 二阶连续可微，则可通过考察以下两个量来判断某点 x∗x^*x∗ 是否为局部极小值点（甚至严格局部极小值点）：

      • 梯度 ∇f(x∗)\nabla f(x^*)∇f(x∗)
      • Hessian 矩阵 ∇2f(x∗)\nabla^2 f(x^*)∇2f(x∗)

    • 一阶必要条件

      • 若 x∗x^*x∗ 是函数 fff 的局部极小值点，且 fff 在 x∗x^*x∗ 的某个开邻域内连续可微，则必有

        ∇f(x∗)=0.\nabla f(x^*) = 0. ∇f(x∗)=0.

        此条件称为一阶必要条件（first-order necessary condition）。
      • 若 ∇f(x∗)=0\nabla f(x^*) = 0∇f(x∗)=0，则称 x∗x^*x∗ 为函数 fff 的驻点（或平稳点，stationary point）。
      • 因此，任何局部极小值点都必须是驻点。

    • 二阶必要条件

      • 若 x∗x^*x∗ 是函数 fff 的局部极小值点，且 Hessian 矩阵 ∇2f\nabla^2 f∇2f 在 x∗x^*x∗ 的某个开邻域内连续，则必有：
        • 梯度为零：∇f(x∗)=0\nabla f(x^*) = 0∇f(x∗)=0，
        • Hessian 矩阵半正定：∇2f(x∗)⪰0\nabla^2 f(x^*) \succeq 0∇2f(x∗)⪰0。

      这两个条件合称为二阶必要条件（second-order necessary conditions）。

    • 二阶充分条件

      • 假设 ∇2f\nabla^2 f∇2f 在点 x∗x^*x∗ 的某个开邻域内连续，且满足

      ∇f(x∗)=0,∇2f(x∗)≻0（即 Hessian 矩阵正定）,\nabla f(x^*) = 0, \quad \nabla^2 f(x^*) \succ 0 \quad \text{（即 Hessian 矩阵正定）}, ∇f(x∗)=0,∇2f(x∗)≻0（即 Hessian 矩阵正定）,

      则 x∗x^*x∗ 是函数 fff 的严格局部极小值点。

      此为二阶充分条件（second-order sufficient condition）。

    • 针对凸函数的情况

      • 若 fff 是凸函数，则其任意局部极小值点 x∗x^*x∗ 必为全局极小值点。
      • 进一步，若 fff 可微，则任意驻点（即满足 ∇f(x∗)=0\nabla f(x^*) = 0∇f(x∗)=0 的点）x∗x^*x∗ 都是 fff 的全局极小值点。

  • 算法总览
    对于一般的非线性最优化问题，最典型和常见的求解方法是迭代型数值方法。根据迭代过程中下一迭代点的生成方式，这类方法可分为随机搜索型方法和确定型方法。

    • 随机搜索型方法是一类仿生智能优化算法，主要依赖目标函数值信息，不依赖梯度或解析结构。
      它适用于组合优化问题以及规模较小的连续优化问题。
      常见方法包括：遗传算法、模拟退火算法、蚁群优化、神经网络方法等。

    • 确定型方法根据所利用函数信息的程度，进一步分为：

      • 直接搜索法（又称模式搜索法）
      • 梯度型方法

    • 直接搜索法通过观察目标函数值的变化规律来探测下降方向，并沿该方向寻找更优解。
      它不需要计算梯度，适用于以下情形：

      • 变量维数较低，
      • 约束结构简单，
      • 目标函数形式复杂或不可导，导致梯度难以获取或计算成本高。
        典型算法包括：坐标轮换法、Hooke-Jeeves 方法、Powell 共轭方向法等。

  • 两种策略
    对于梯度型数值方法(Gradient-based),一般通过两种策略，来
    由当前迭代点产生下一选代点：线搜索策略和 信赖域策略。

    • 线搜索
      • 在线搜索策略（line search）中，算法首先选择一个搜索方向 pkp_kpk​，然后从当前迭代点 xkx_kxk​ 出发，沿该方向寻找一个函数值更低的新迭代点。
      • 沿方向 pkp_kpk​ 移动的步长 α>0\alpha > 0α>0 通常通过近似求解如下一维优化问题来确定：

        min⁡α>0f(xk+αpk).\min_{\alpha > 0} f(x_k + \alpha p_k). α>0min​f(xk​+αpk​).

      • 虽然精确求解该一维问题能最大程度地利用方向 pkp_kpk​ 的下降潜力，但精确线搜索计算代价高且通常不必要。因此，实际算法多采用满足一定条件（如 Wolfe 条件）的非精确线搜索。
    • 信赖域
      • 在信赖域策略（trust region method）中，算法利用当前已知的关于目标函数 fff 的信息，构造一个模型函数 mkm_kmk​，使其在当前迭代点 xkx_kxk​ 附近的行为与真实的 fff 相似。
      • 然而，当 xxx 远离 xkx_kxk​ 时，模型 mkm_kmk​ 可能不再是对 fff 的良好近似。
      • 因此，该策略将对模型函数的最小化限制在 xkx_kxk​ 周围的一个信赖域内，即求解如下子问题：

      min⁡pmk(xk+p)s.t.∥p∥2≤Δk,\begin{aligned} \min_{p} \quad & m_k(x_k + p) \\ \text{s.t.} \quad & \|p\|_2 \leq \Delta_k, \end{aligned} pmin​s.t.​mk​(xk​+p)∥p∥2​≤Δk​,​

      其中 Δk>0\Delta_k > 0Δk​>0 是当前信赖域的半径，用于控制搜索范围。
      • 模型函数 mkm_kmk​ 通常取为二次函数，形式如下：

        mk(xk+p)=fk+pT∇fk+12pTBkp,m_k(x_k + p) = f_k + p^T \nabla f_k + \frac{1}{2} p^T B_k p, mk​(xk​+p)=fk​+pT∇fk​+21​pTBk​p,

        其中：
        • fk=f(xk)f_k = f(x_k)fk​=f(xk​)，
        • ∇fk=∇f(xk)\nabla f_k = \nabla f(x_k)∇fk​=∇f(xk​) 是目标函数在 xkx_kxk​ 处的梯度，
        • BkB_kBk​ 是 Hessian 矩阵 ∇2f(xk)\nabla^2 f(x_k)∇2f(xk​) 或其某种近似（如拟牛顿法中的近似）。
    • 线搜索VS信赖域

      • 线搜索（Line Search）与信赖域（Trust Region）方法的核心区别在于确定方向与步长的顺序不同。

      • 线搜索方法：

        • 首先选定一个搜索方向 pkp_kpk​；
        • 然后在该方向上确定一个合适的步长 αk>0\alpha_k > 0αk​>0，使得新点 xk+1=xk+αkpkx_{k+1} = x_k + \alpha_k p_kxk+1​=xk​+αk​pk​ 能有效降低目标函数值。

      • 信赖域方法：

        • 首先设定一个最大移动距离，即信赖域半径 Δk>0\Delta_k > 0Δk​>0；
        • 然后在球形区域 {p∣∥p∥2≤Δk}\{ p \mid \|p\|_2 \leq \Delta_k \}{p∣∥p∥2​≤Δk​} 内同时选择方向和步长（即向量 ppp），以在该约束下尽可能优化模型函数，从而获得最佳改进。

  • 线搜索——搜索方向

    • 线搜索算法要求搜索方向 pkp_kpk​ 是一个下降方向（descent direction），即满足

    pkT∇f(xk)<0.p_k^T \nabla f(x_k) < 0. pkT​∇f(xk​)<0.

    • 这一条件保证了：存在某个 ε>0\varepsilon > 0ε>0，使得对所有 0<α≤ε0 < \alpha \leq \varepsilon0<α≤ε，有

    f(xk+αpk)=f(xk)+α pkT∇f(xk)+o(α)<f(xk),f(x_k + \alpha p_k) = f(x_k) + \alpha\, p_k^T \nabla f(x_k) + o(\alpha) < f(x_k), f(xk​+αpk​)=f(xk​)+αpkT​∇f(xk​)+o(α)<f(xk​),

    即沿 pkp_kpk​ 走“足够小”的一步，函数值必定下降。

    • 一旦确定了下降方向，下一步是通过线搜索确定合适的步长 αk\alpha_kαk​。
      通常，这类算法生成的迭代点列 {xk}\{x_k\}{xk​} 对应的目标函数值 {f(xk)}\{f(x_k)\}{f(xk​)} 是单调递减的，因此线搜索方法也被称为下降算法。

    • 在影响算法效率的因素中，搜索方向的选择比步长的精确性更为关键。一个良好的方向往往比精细调整步长更能加速收敛。

    • 搜索方向包括：

      • 最速下降方向：pk=−∇f(xk)p_k = -\nabla f(x_k)pk​=−∇f(xk​)
      • 牛顿方向：pk=−[∇2f(xk)]−1∇f(xk)p_k = -[\nabla^2 f(x_k)]^{-1} \nabla f(x_k)pk​=−[∇2f(xk​)]−1∇f(xk​)
      • 拟牛顿方向：pk=−Bk−1∇f(xk)p_k = -B_k^{-1} \nabla f(x_k)pk​=−Bk−1​∇f(xk​)，其中 BkB_kBk​ 是 Hessian 矩阵的近似

  • 最速下降方向

    • 最速下降方向（steepest descent direction）是指在当前点 xkx_kxk​ 处，使目标函数 fff 下降最快的方向。

    • 考虑一阶泰勒展开：

    f(xk+αp)=f(xk)+α pT∇f(xk)+o(α),α>0.f(x_k + \alpha p) = f(x_k) + \alpha\, p^T \nabla f(x_k) + o(\alpha), \quad \alpha > 0. f(xk​+αp)=f(xk​)+αpT∇f(xk​)+o(α),α>0.

    要使函数值在单位步长下下降最多，应最小化方向导数 pT∇f(xk)p^T \nabla f(x_k)pT∇f(xk​)，即求解：

    min⁡p  pT∇fksubject to∥p∥=1.\min_{p} \; p^T \nabla f_k \quad \text{subject to} \quad \|p\| = 1. pmin​pT∇fk​subject to∥p∥=1.

    • 该问题的解为：

    p=−∇fk∥∇fk∥,p = -\frac{\nabla f_k}{\|\nabla f_k\|}, p=−∥∇fk​∥∇fk​​,

    即负梯度方向是单位长度下的最速下降方向。

    • 因此，最速下降法（又称梯度法或梯度下降法）采用搜索方向：

    pk=−∇f(xk).p_k = -\nabla f(x_k). pk​=−∇f(xk​).
